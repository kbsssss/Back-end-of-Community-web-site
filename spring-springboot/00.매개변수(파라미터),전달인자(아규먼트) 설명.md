## 매배견수(파라미터) 와 전달인자(아규먼트)의 개념

* 매개변수(=paramaeter,파라미터)
    메서드의 public in sum(int a, int b){
        ~
        }
        와 같이 되있을때, int a, 와 int b를
        매개변수 혹은 파라미터라고 부른다.
        
* 전달인자(=Argument, 아규먼트)
    실제로 위의 메서드를 사용할때, sum(10,20);
    로 메서드를 호출하는데, 이 실질적인 값 10,20을
    전달인자 혹은 아규먼트라고 한다.
